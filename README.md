# Python Mini Projects

A collection of small Python programs built while practicing core concepts such as loops, functions, file handling, and basic logic.

## Projects

| Project | Description |
|---|---|
| Choose Your Own Adventure | Text-based branching story game |
| Madlibs Generator | Generates a silly story from user word input, using story.txt as a template |
| Number Guessing Game | Guess-the-number game with attempt tracking |
| Password Manager | Simple CLI tool to store and retrieve passwords |
| Pig Dice Game | Two-player dice game with turn-based scoring |
| Quiz Game | Multiple choice quiz with score tracking |
| Rock Paper Scissors | Classic game against the computer |

More projects will be added here as I continue practicing.
